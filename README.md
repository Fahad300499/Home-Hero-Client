<h1>5 Key Features:</h1>
<li> Dual-Role Authentication and Authorization: Robust Firebase-based login/registration system supporting two user roles (Service Providers and Customers), with personalized dashboards and private routes (My Services, My Bookings) secured via Firebase Token Verification.</li>
<li>Comprehensive Service Management (CRUD): Service Providers have full Create, Read, Update, and Delete capabilities to manage their service listings, including price, category, and image details, stored securely in MongoDB.</li>
<li>Secure Booking and Appointment System: Customers can seamlessly book services via a confirmation modal, and manage their appointments through the 'My Bookings' page, including a quick option to cancel (DELETE) bookings.</li>
<li>Advanced Dynamic Data Features: Implements MongoDB operators for dynamic price filtering ($gte, $lte) on the service list page, and a robust search feature for filtering by name or category.</li>
<li>Integrated Rating and Review System: Allows customers to submit ratings and reviews for booked services, which are stored within the service object, contributing to service popularity and visibility (e.g., top-rated services on the homepage).</li>
